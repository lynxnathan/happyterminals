    Implement `TachyonAdapter<S: tachyonfx::Shader>`:
    ```rust
    use std::time::Duration;
    use tachyonfx::Shader;
    use ratatui_core::layout::Rect;
    use happyterminals_core::Grid;
    use crate::effect::{Effect, EffectState};

    /// Adapts any tachyonfx [`Shader`] into our [`Effect`] trait (D-04).
    ///
    /// Duration conversion: our `std::time::Duration` dt is converted to
    /// `tachyonfx::Duration` (a `{milliseconds: u32}` struct) via `From`.
    /// The adapter passes `grid.buffer_mut()` to the shader's `process()`.
    #[derive(Debug)]
    pub struct TachyonAdapter<S: Shader> {
        shader: S,
        area: Option<Rect>,  // None = use full grid area
        completed: bool,
    }
    ```

    - Constructor: `TachyonAdapter::new(shader: S) -> Self` (area=None, full grid)
    - Constructor: `TachyonAdapter::with_area(shader: S, area: Rect) -> Self` (custom area)
    - `Effect::apply()`: Convert `dt` via `tachyonfx::Duration::from(dt)`, call `self.shader.process(tfx_dt, grid.buffer_mut(), area)`, check `self.shader.done()`, return `EffectState::Running` or `Done`.
    - `Effect::is_done()`: return `self.completed`
    - `Effect::reset()`: call `self.shader.reset()`, set `self.completed = false`
    - `Effect::name()`: delegate to `self.shader.name()`

    CRITICAL: The area defaults to `grid.area` (accessed via `Deref` to Buffer). If `self.area` is `Some(rect)`, use that instead. This handles Pitfall 6.

    CRITICAL: Convert duration via `tachyonfx::Duration::from(dt)` — tachyonfx Duration is `{milliseconds: u32}`, NOT `std::time::Duration` (Pitfall 1).

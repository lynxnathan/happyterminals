## TachyonAdapter internals

| Option | Description | Selected |
|--------|-------------|----------|
| Pass Grid via inner_mut() | TachyonAdapter calls grid.inner_mut() to get &mut Buffer for tachyonfx | |
| Pass Grid via Deref + separate mutation | Read via Deref, mutate via a new public method | |
| Pass Grid via conversion | Convert Grid → Buffer, run effect, convert back | ✓ (auto: simplest, avoids pub(crate) issue) |

**Auto-selected:** Grid derefs to Buffer for reads; for writes, TachyonAdapter needs mutable Buffer access. Recommended: expose a `pub fn as_buffer_mut(&mut self) -> &mut Buffer` on Grid for cross-crate effect application.
**Notes:** The `inner_mut()` method is pub(crate) and cannot be called from the pipeline crate. A public method is the cleanest solution.

## Pipeline ordering

| Option | Description | Selected |
|--------|-------------|----------|
| Sequential (Vec order) | Each effect applies after the previous, receiving the mutated Grid | ✓ (auto: recommended) |
| Parallel (blend) | Effects apply to copies of Grid, then blend | |
| Configurable | Per-effect flag for sequential vs parallel | |

**Auto-selected:** Sequential — simplest, composable, sufficient for M1. Parallel blending is complex and not needed for the spinning cube demo.

## EffectState lifecycle

| Option | Description | Selected |
|--------|-------------|----------|
| Skip when done, reset() for reuse | Done effects stay in Vec but are skipped. reset() re-enables them. | ✓ (auto: recommended) |
| Auto-remove when done | Done effects are removed from the Vec automatically | |
| Callback on done | Fire a callback/signal when effect finishes | |

**Auto-selected:** Skip + reset — stable Vec indices, no allocation churn, supports looping via reset().

## Grid mutation invariant enforcement

| Option | Description | Selected |
|--------|-------------|----------|
| Signature-enforced | Grid only accessible via &mut Grid in apply() signature | ✓ (auto: recommended) |
| Clippy lint | disallowed_methods on direct Grid mutation | |
| Runtime check | Debug assertion that Grid is only mutated during apply() | |

**Auto-selected:** Signature enforcement is the strongest guarantee — no runtime cost, compiler catches violations.

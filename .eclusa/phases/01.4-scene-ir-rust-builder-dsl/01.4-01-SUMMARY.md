## Decisions Made
- Removed Debug bound from AnyReactive trait because Signal<T> and Memo<T> do not implement Debug in the existing codebase
- PropValue uses Box<dyn Any> (no Send+Sync) per research Pitfall 1 -- D-05 mentions Send+Sync but that applies to the JSON/Python path, not the Rust builder path
- Scene::new is pub (not pub(crate)) so the DSL crate can call it from SceneBuilder::build()
- TransitionManager.take() uses &mut self (not consuming self) for flexibility

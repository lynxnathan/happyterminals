## Decisions Made
- Used generic return `R: Into<LayerBuilder>` on layer()/group() closures instead of `impl IntoLayer` trait (not stable in closure args) -- enables `|l| l.cube().position(v)` to return CubeBuilder which auto-converts
- Created separate GroupCubeBuilder type (not just CubeBuilder) because group cubes need a GroupBuilder parent, not LayerBuilder
- Named the method `pipeline()` instead of `effect()` since it accepts a full Pipeline, not individual effects
- Hello world test uses DSL prelude (not meta crate prelude) to avoid circular dev-dependency; meta prelude tested via `cargo test -p happyterminals --lib`

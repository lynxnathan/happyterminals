    - `pub ramp: &'a [char]` with `DEFAULT_RAMP: &[char] = &[' ', '.', ':', '-', '=', '+', '*', '#', '%', '@']`

    - `pub fn view_matrix(&self) -> Mat4`: Convert spherical to Cartesian eye position:
      ```
      eye.x = target.x + distance * elevation.cos() * azimuth.sin()
      eye.y = target.y + distance * elevation.sin()
      eye.z = target.z + distance * elevation.cos() * azimuth.cos()
      ```
      Then `Mat4::look_at_rh(eye, self.target, Vec3::Y)`.
    - Implement `Default` with azimuth=0.0, elevation=0.0, distance=5.0, target=Vec3::ZERO.
    - Note: Signal-driven camera is achieved by reading `signal.untracked()` into these f32 fields before calling view_matrix(). The Renderer in Plan 02 handles this. Camera struct itself is pure data, no Signal dependency.

  <action>
    **rasterizer.rs** — Implement per D-04 (half-space edge functions) and RESEARCH Pattern 2 and 5:

    1. `fn edge_function(v0x, v0y, v1x, v1y, px, py) -> f32` — inline, returns signed area.

    2. `fn project_vertex(pos: Vec3, mvp: &Mat4, half_w: f32, half_h: f32) -> Option<(f32, f32, f32)>`:
       - Compute clip = mvp * Vec4(pos, 1.0)
       - If clip.w <= 0.0, return None (behind camera, per Pitfall 5)
       - Perspective divide: ndc_x = clip.x/clip.w, ndc_y = clip.y/clip.w, ndc_z = clip.z/clip.w
       - Screen: screen_x = (ndc_x + 1.0) * half_w, screen_y = (1.0 - ndc_y) * half_h (flip Y)
       - Return Some((screen_x, screen_y, ndc_z))

    3. `fn rasterize_triangle(sv: &[(f32, f32, f32); 3], z_buffer: &mut [f32], grid_w: usize, grid_h: usize, shade_char: char, cell_chars: &mut [char])`:
       - Compute bounding box clamped to grid dimensions
       - For each cell in bbox: compute edge functions w0, w1, w2
       - If all >= 0.0: barycentric depth interpolation, reversed-Z depth test (depth > z_buffer[idx] per D-03)
       - Write shade_char to cell_chars[idx]

    4. `fn backface_cull(face_normal: Vec3, camera_dir: Vec3) -> bool`:
       - Returns true if face should be culled (normal dot camera_dir >= 0.0, face points away)

    **lib.rs** — Add `pub mod rasterizer;` and implement `Renderer` struct per D-10, D-11, D-12, D-13:

    ```rust
    pub struct Renderer {
        z_buffer: Vec<f32>,
        cell_chars: Vec<char>,  // staging buffer, same size as z_buffer
        last_width: u16,
        last_height: u16,
    }
    ```

    `pub fn draw(&mut self, grid: &mut Grid, camera: &OrbitCamera, projection: &Projection, shading: &ShadingRamp)`:
    1. Get (w, h) from grid.area via Deref
    2. Resize z_buffer and cell_chars ONLY if dimensions changed (zero-alloc after warmup per D-12)
    3. Clear z_buffer with fill(0.0) (reversed-Z far), cell_chars with fill(' ')
    4. Compute view = camera.view_matrix(), proj = projection.matrix(), mvp = proj * view
    5. Compute camera_dir = (camera.target - eye).normalize() for backface culling
    6. For each face (0..6), pair of triangles (face*2, face*2+1):
       a. Backface cull using FACE_NORMALS[face] dot camera_dir
       b. Project 3 vertices via project_vertex; if any returns None, skip triangle
       c. Compute shade_char = shading.shade(FACE_NORMALS[face])
       d. rasterize_triangle into z_buffer + cell_chars
    7. Write cell_chars into Grid via `grid.buffer_mut().cell_mut(Position::new(x, y)).set_char(ch)` for each non-space character

    Scene-fit z_near per D-05: For M1 with a unit cube at known distance, compute z_near from bounding sphere. Add a helper `fn scene_fit_near(camera: &OrbitCamera) -> f32` that returns `(camera.distance - CUBE_BOUNDING_RADIUS).max(0.01)` where CUBE_BOUNDING_RADIUS = sqrt(3)/2 ~= 0.866. Set this on Projection before computing the matrix, or pass it to Projection::matrix_with_near(z_near).

    Unit tests in rasterizer.rs for edge_function, project_vertex, and backface_cull.

## Decisions Made
- Cell is a zero-sized utility namespace rather than a wrapping struct. Since ratatui's Buffer stores its own Cell type and Grid Derefs to Buffer, wrapping would add indirection with no benefit. Users read cells directly via `grid.cell(pos)`.
- Grid exposes `inner_mut()` as `pub(crate)` for future backend blit access. This will need to become `pub` or use a different access pattern when the backend crate consumes it (tracked for Plan 02).
- Clippy truncation warning on `grapheme.width() as u16` fixed with `u16::try_from().ok()` -- grapheme widths are always small but the cast is technically fallible.

         IMPORTANT: The existing `draw()` method has its own Grid-write loop. Do NOT modify it -- add a SEPARATE Grid-write section inside `draw_particles()` that writes only the particle cells (where cell_colors is Some). This avoids breaking the existing mesh rendering path.

**Pool-based particle system with Copy struct, z-buffered point projection, color-over-time lerp, and criterion bench enforcing zero per-frame allocation across all 3 renderer paths (cube, OBJ mesh, particles)**

- Criterion bench covers all 3 renderer paths (cube 12-tri, bunny ~5k-tri, particles 500-pool) with capacity-stability assertions

## Decisions Made
- Used rand (workspace dep) for spawn randomization -- already in dep tree, no new transitive deps

- Added public capacity accessor methods on Renderer for bench assertions (cleaner than pub fields)
- draw_particles has its own Grid-write section separate from draw() to avoid breaking existing mesh rendering

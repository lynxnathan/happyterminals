## Nice-to-have improvements (advisory)

1. **Plan 03 Task 1 snapshot scene determinism**: the tests rely on the renderer producing byte-identical output across runs. RESEARCH and the plan note this is achieved via fixed viewport + hardcoded camera + no RNG/clock ‚Äî but executor should run the test 3√ó back-to-back (not just twice as `<done>` requires) before the first commit, to catch any latent non-determinism in the renderer's iteration order. Cheap insurance.

2. **Plan 02 Task 1 `with_mode_uninitialized_for_test` seam**: the `#[cfg(test)] uninitialized: bool` field on `TerminalGuard` is a clean test seam, but the action block already acknowledges a fallback to `ManuallyDrop` if cfg-gated fields conflict with `Drop`. Either path is fine; executor should pick whichever compiles first without modifying the contract. Consider documenting the chosen approach in the SUMMARY for future maintainers.

3. **Plan 03 Task 2 `--force-color` arg parser**: the parser accepts only the four exact strings. Consider whether `--force-color none` (space-separated) or `--no-color` aliases would be friendly ‚Äî but this is genuinely deferred surface, not an in-scope gap. Current strict matching matches the planning constraint.

4. **VALIDATION manual UAT row 4** (`NO_COLOR=1` + `--force-color=truecolor` ‚Üí Mono): this is also covered by `no_color_beats_override` unit test. The manual UAT is a belt-and-suspenders gate at the real-binary level ‚Äî explicitly justified in VALIDATION as such. No action needed; just noting that this gate has both unit and manual coverage, not a redundancy bug.

5. **Plan 02 Task 1 `acquire()` shim defaults to `TrueColor`**: this preserves the API for legacy callers but means a caller using the old `acquire()` skips the `downsample` fast-path opportunity. Since nothing in-tree calls `acquire()` outside the Plan 02-modified `run`/`run_scene` (which now call `acquire_with_color_mode`), this is purely a back-compat shim ‚Äî fine. Worth a one-line comment in the SUMMARY noting that the shim path will be removed in a future major if no downstream depends on it.

## Decisions Made

See frontmatter `key-decisions`. Most consequential:

1. **Retain `acquire()` as a TrueColor shim** — prevents churn across ~18 call sites in backend tests. New code should prefer `acquire_with_color_mode(mode)`.
2. **`#[cfg(test)]` uninitialized field in `TerminalGuard`** — localizes the TTY-free test seam to the guard itself. Production `TerminalGuard` has exactly the 2-field layout the plan's `<interfaces>` promised (`stdout`, `color_mode`); the uninitialized flag is invisible to non-test builds.
3. **Downsample fires post-pipeline in `run_scene`** — effect stacks can still emit truecolor; the single downsample locus enforces the color-mode contract uniformly.

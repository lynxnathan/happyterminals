## human_verification

Manual UAT matrix to run before closing the phase. Total expected runtime: ~2 minutes. Run from the workspace root.

### 1. Launch + visual recognizability

**Command:**
```
cargo run --example model-viewer -p happyterminals
```

**Expected:**
- Terminal enters alt-screen with title bar showing `happyterminals - Model Viewer`
- Row 0 overlays ` Model: bunny  (Left/Right to cycle, Ctrl-C to exit) `
- Below row 0: a slowly-orbiting ASCII silhouette that is recognizably a **bunny** (upright ears, crouched body, hind legs folded under — Stanford bunny pose)
- No black faces, no garbled blocks, no NaN artifacts, no all-same-character fill

### 2. Arrow-key cycling

**Action:** Press `Right` arrow once.

**Expected:**
- Row 0 label changes to `Model: cow`
- Silhouette transitions to a recognizably **cow**-shaped form (four legs, horizontal body, tail / horns visible depending on the OBJ detail)
- Auto-fit keeps the cow filling a similar fraction of the viewport as the bunny did

**Action:** Press `Right` arrow once more.

**Expected:**
- Row 0 label changes to `Model: teapot`
- Silhouette is the Utah teapot (handle + spout visible)

**Action:** Press `Right` arrow once more.

**Expected:**
- Wraps back to `Model: bunny`

**Action:** Press `Left` arrow once.

**Expected:**
- Cycles backwards: `Model: teapot` (wraps down through zero)

### 3. Auto-fit camera spot-check

**Action:** Cycle through all three models once.

**Expected:**
- Each model fills roughly **60% of the viewport area** — no model appears as a tiny dot in the center or clips off the edges
- Vertical framing is consistent across models

### 4. Responsiveness

**Action:** Mash `Right` arrow 10× rapidly.

**Expected:**
- Every press registers (label cycles visibly with each press)
- No dropped events (model name matches expected cycle count: bunny → cow → teapot → bunny → …)
- No visible stutter / frame drop during rapid cycling

### 5. Clean exit

**Action:** Press `Ctrl-C`.

**Expected:**
- Terminal returns to main screen buffer
- Cursor is visible and positioned at a sane row
- Raw mode is off (typing `echo hello` after exit produces normal echoed output)
- No leftover ANSI state (colors, SGR, title bar all reset)

Any failure of items 1-5 should be reported as a gap and the verification status flipped to `gaps_found`. If all five pass, the verifier can update this file's frontmatter `status` to `passed` and the phase advances to 2.2.

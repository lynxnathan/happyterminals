## Decisions Made
- run_with_input() kept as separate function (not modifying existing run()) to preserve backward compat
- InputMap dispatch happens before legacy signal path in event arm so InputMap sees all raw events
- reset_axes() called after render_fn (at end of tick block) so render reads accumulated deltas for the frame
- Model cycling uses action system with JustPressed checks instead of hardcoded key matching
- Auto-fit camera distance only resets when model index changes, preserving user zoom adjustments

         ```rust
         pub struct InputMap {
             contexts: Vec<InputContext>,  // index 0 = bottom ("default"), last = top
             actions: std::collections::HashMap<String, ActionEntry>,
             drag_machine: DragStateMachine,
         }
         ```
       - `impl InputMap`:
         - `pub fn new() -> Self` with empty contexts vec, empty actions, fresh DragStateMachine.
         - `pub fn register_action(&mut self, key: impl Into<String>, value_type: ActionValueType)` inserts `ActionEntry::new(value_type)` into actions map.
         - `pub fn push_context(&mut self, context: InputContext)` appends to contexts vec.
         - `pub fn pop_context(&mut self, name: &str) -> Option<InputContext>`: find by name with `rposition`, refuse to pop index 0, remove and return.
         - `pub fn dispatch(&mut self, event: &crossterm::event::Event)`:
           Phase 1: if Mouse event, update drag_machine, capture DragOutput.
           Phase 2: iterate `self.contexts.iter().rev()` -- for each context, call `ctx.try_resolve(event, drag_output.as_ref())`. On first `Some(fired)`, call `self.apply_action(&fired)` and return.
         - `fn apply_action(&self, fired: &FiredAction)`:
           Look up action entry by `fired.action_key`. Apply modifier chain via `apply_chain(&fired.modifiers, fired.raw_value.clone())`. Then set the appropriate signal:
           - `ActionValue::Bool(b)` -> set state to `JustPressed` if true, `Released` if false.
           - `ActionValue::Axis1D(v)` -> set axis1d signal if present.

         - `pub fn action_axis1d(&self, key: &str) -> Option<&Signal<f32>>` returns axis1d ref.
         - `pub fn action_axis2d(&self, key: &str) -> Option<&Signal<glam::Vec2>>` returns axis2d ref.

         - `pub fn rebind(&mut self, context_name: &str, action: &str, binding: Binding)`: find context by name, call `ctx.rebind(action, binding)`.

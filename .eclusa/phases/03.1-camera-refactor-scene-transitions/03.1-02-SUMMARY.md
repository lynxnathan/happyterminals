## Decisions Made
- Used buffer-level blending (not Grid-level) per RESEARCH pitfall avoidance -- Grid restricts write access
- Dissolve uses hard cutoff at progress 0.5 (not alpha blending) since terminals have no transparency
- FadeToBlack wipes through black left-to-right using column-ratio threshold, creating cinematic feel
- Added `take()` method to TransitionManager for backward compatibility with existing integration tests
- Added `UnknownEffect` and `NotIdle` error variants to SceneError for proper error handling

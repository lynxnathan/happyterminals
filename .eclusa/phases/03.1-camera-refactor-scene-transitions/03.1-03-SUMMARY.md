## Decisions Made
- Used cube scenes with different camera angles rather than mesh/particle scenes, because the scene graph IR only supports NodeKind::Cube currently. Adding Mesh and Particle node kinds would be an architectural change (Rule 4) outside this plan's scope.
- Tab detection uses legacy InputSignals.last_key in the tick_fn callback rather than InputMap action_state, because the InputMap dispatch happens in run_scenes before tick_fn is called, making action_state already transitioned to Held by the time tick_fn reads it. The legacy signal path provides the raw key event.
